\section{团队角色}

Evensong 是 DASH SHATTER Research 的核心评估系统。它在记忆条件和压力水平下测试 AI 代理，为研究出版物和基准数据集生成行为画像。

\textbf{团队职能：}工程 + 研究（混合角色）

\section{我们需要的人才技能}

\subsection*{核心 Technical}
\begin{itemize}
  \item \textbf{Bun + TypeScript：}工具链运行在 Bun 上。需要对 TS 接口和异步模式熟悉
  \item \textbf{LaTeX / pgfplots：}用于论文和报告的文档生成
  \item \textbf{CLI 工具：}Shell 脚本、进程管理、管道
  \item \textbf{多模型 API 路由：}OpenRouter、EverOS、模型提供商配置
\end{itemize}

\subsection*{研究 fluency}
\begin{itemize}
  \item \textbf{实验设计：}2$\times$2 因子矩阵、受控条件
  \item \textbf{基准测试方法论：}熟悉 SWE-bench、LiveCodeBench、Berkeley FCF
  \item \textbf{压力校准：}PUA 级别（L0-L3）、提示注入、行为测量
\end{itemize}

\section{面试流程}

\textbf{第一阶段（45 分钟）：}给定基准测试工具链设置，在 3 次运行中识别记忆隔离 bug。观察他们如何处理问题。

\textbf{第二阶段（60 分钟）：}使用新模型运行一次基准测试。评估：候选人是否 catch 到 API 配置错误？

\textbf{第三阶段（30 分钟）：}讨论自我修复分类法。探究：他们如何扩展 4 类分类？

\section{薪酬基准}

Evensong 经验稀缺的工程师深入理解 AI 行为的机制层面——对研究工程、AI 安全和前沿模型评估角色非常有价值。
