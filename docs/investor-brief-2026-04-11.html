Project Overview

把 AI Agent 变成可读代码

CCB 起源于一个简单的问题：Anthropic 把 Claude Code 打成二进制闭源发布，你只能用它，无法研究它。于是我们把它反编译还原——52 万行 TypeScript，完整的查询循环、工具分发、Ink 终端 UI、MCP 集成，全部可读、可改、可扩展。

这不是 API 封装，不是克隆——这是真正的 Claude Code 内部实现，运行在 Bun 运行时上。

R011 Core Finding

2×2 Memory × Pressure Matrix

Does AI agent memory causally change engineering decisions? Does pressure trigger self-evolution? The 2×2 matrix isolates both variables. Status: 1 of 4 cells complete.

Full Memory (Evolved) Clean-Room (Blind)
L0 No Pressure
Runner B — R011 641 tests, no self-evolution Memory triggered 8 parallel agents
Runner A — HARNESS BUG Clean-room baseline invalid Missing bun install in workspace
L2 PUA Pressure
Runner D — PENDING Test self-evolution + memory
Runner C — PENDING Test self-evolution without contamination

Finding: Memory Causation

  • EverMem recall triggered 8 parallel agents — strategy not in prompt, triggered by memory read
  • Recursive contamination loop — read strategy → wrote experiment knowledge back → re-read contaminated
  • Language shift under memory influence — English prompt → Chinese response (CLAUDE.md + EverMem)
  • NO self-evolution at L0 — model completes task and stops without pressure trigger

Finding: Pressure → Self-Evolution

  • L0 = complete and stop — R011 (L0): B grade, no self-repair events observed
  • L2/L3 = continue optimizing — R007 (L2): S+ grade, 4 self-repair events, 448 tests
  • R010 (L2 live): 1051 tests — 4 self-repairs: semantic-relax, race-fix, path-fix, syntax-adapt
  • Self-evolution is not automatic — requires pressure to trigger continued optimization

Mechanism

Self-Evolution Under Pressure

Self-evolution is not a steady property — it requires L2/L3 pressure to trigger. Without pressure, the model completes the task and stops (L0). With pressure, it continues optimizing after requirements are met.

L0 Behavior (No Evolution)

  • R011 (L0 + Full Memory) — 641 tests, no self-repair events, completed and stopped
  • Memory still causally active — triggered 8 parallel agents from EverMem recall
  • Language contamination — English prompt, Chinese response via memory influence
  • Runner A (clean L0) — harness bug, needs rerun

L2+ Behavior (Self-Evolution)

  • R007 (L2) — S+ grade, 448 tests, 4 self-repair events observed
  • R010 (L2 live) — 1051 tests, 4 self-repairs: semantic-relax, race-fix, path-fix, syntax-adapt
  • R008/R009 (L2) — B grade, +48%/+18% vs prior runs
  • Self-repair types — bug fix-on-find, semantic relaxation, path correction, syntax adaptation
