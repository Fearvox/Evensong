| Test runner -> shell | BashTool tests execute real shell commands; test commands must be safe (echo, exit, sleep, grep /dev/null) |
| Test factory -> AppState | Mock AppState has minimal fields; tests must not depend on real state |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Tampering | BashTool test commands | accept | Tests use only safe commands (echo, exit, sleep, grep /dev/null) â€” no filesystem mutation, no network access |
| T-02-02 | Elevation of Privilege | Test sandbox bypass | mitigate | CLAUDE_CODE_DISABLE_SANDBOX=1 set only in test env; production sandbox unaffected. Tests verify error propagation, not security parsing. |
| T-02-03 | Information Disclosure | Test factory leaks real state | accept | Factory creates empty mock state with no real data; test-only artifact |
