   - 用 `tokenUsage = threshold`（精确等于阈值）
   - 断言 `isAboveAutoCompactThreshold === true`

   - 断言 `isAboveAutoCompactThreshold === true`

4. **isAutoCompactEnabled 禁用时不触发**：
   - 设置 `DISABLE_AUTO_COMPACT = '1'`
   - 任意 `tokenUsage = threshold + 10000`（远超阈值）
   - 断言 `isAboveAutoCompactThreshold === false`（因为 isAutoCompactEnabled() 返回 false）

5. **DISABLE_COMPACT 完全禁用**：
   - 设置 `DISABLE_COMPACT = '1'`
   - 调用 `isAutoCompactEnabled()`

   - 断言 `isAtBlockingLimit === true`
   - 用 `tokenUsage = 4999`，断言 `isAtBlockingLimit === false`

7. **percentLeft 计算合理**：
   - 不设置 DISABLE_AUTO_COMPACT（autocompact enabled）
   - 用 `tokenUsage = 0`，断言 `percentLeft === 100`
   - 用 `tokenUsage = threshold`，断言 `percentLeft === 0`（或 ≤ 1，因为 Math.round）
